Androdex Waitlist

WaitListに登録して
Androidアプリの案内を受け取る

Androdex は、Windows 上で動くローカル Codex を Android から操作するための independent fork です。

現在の Android 版は、source build で先に試すか、後日の Google Play 配布に向けて waitlist に登録する形で案内しています。

English: waitlist.html

試し方を選ぶ

waitlist に登録する

送信内容は Google Apps Script 経由でスプレッドシートに記録されます。

source build ですぐ試したい方向けの案内です

Android アプリを source build し、Windows 側で androdex up を実行して、Android からペアリングしてください。

日本語 README の build 手順 https://github.com/Ranats/androdex/blob/main/README.ja.md GitHub Issues で報告する https://github.com/Ranats/androdex/issues X で連絡する https://x.com/Ranats85

Play waitlist に登録しました

後日の Play 配布に向けて、メールアドレスを記録しました。Play 側の準備ができたら、Google Group や Play opt-in の案内に進めます。

Google Group を開く https://groups.google.com/g/androdex-android-testers リポジトリに戻る https://github.com/Ranats/androdex X で更新を見る https://x.com/Ranats85

先に Google Groups で androdex-android-testers を検索して参加しておいても大丈夫です。

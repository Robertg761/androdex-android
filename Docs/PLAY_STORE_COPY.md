# Play Store Copy Templates

These templates are written for a paid, buy-once Android app release of Relaydex.

## Short description

```text
Control local Codex on your Windows PC from Android.
```

## Full description

```text
Relaydex lets you control local Codex running on your own computer from Android.

This app is built for a local-first workflow:
- Codex runs on your Windows PC or other host machine
- Android acts as a paired remote control client
- prompts, streaming replies, and approval actions happen through the paired bridge session

What you can do:
- pair with a local host using a QR code
- paste the pairing payload if needed
- browse existing threads
- open or create conversations
- send prompts to local Codex
- receive streaming output
- answer approval requests from Android

Important:
- Relaydex is an independent fork of Remodex
- this is not the official Remodex app
- this app does not run Codex on the phone itself
- you need a compatible local host bridge and a working Codex setup on your own computer
```

## Store listing notes

- Keep `independent fork of Remodex` in the long description.
- Do not present the app as official Remodex or official OpenAI software.
- Show Android screenshots only.
- Make the host requirement clear in the first screenful of text.

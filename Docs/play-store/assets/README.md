## Phone screenshots

All screenshots in `phone-screenshots/` are portrait PNGs at `1440x2560` (`9:16`).
Each file is well under Google Play's `8 MB` limit, and the curated set avoids screenshots that show error dialogs, failed relay states, or expired host status.

Suggested upload order:

1. `01-home-connected.png` — paired host overview and project/thread landing state
2. `02-thread-ready.png` — active conversation view with a clean completed exchange
3. `03-runtime-status.png` — composer state with quick runtime controls
4. `04-project-switching.png` — sidebar and project/thread switching

## Relaydex Android closed test

Relaydex is an independent fork of Remodex focused on controlling local Codex running on Windows from Android.

This is not the official Remodex app.

## What the app does

- pair with your host machine using a QR code
- open and create Codex threads from Android
- send prompts to local Codex
- receive streaming output
- handle approval requests from your phone

Important:

- the app does not run Codex on the phone itself
- Codex still runs locally on your own computer
- you need an Android device and a Google account
